## 🎯 项目背景

### 项目目标
开发一款基于 NFC 技术的"碰一碰组队"应用，用户可以通过 NFC 标签快速组队、加入团队。

### 核心功能
- NFC 标签读取/写入
- 创建/加入队伍
- 实时聊天
- 位置共享
- 队伍管理

### 技术选型
- **框架**: React Native 0.73.6
- **语言**: TypeScript
- **状态管理**: Zustand
- **导航**: React Navigation 6
- **NFC**: react-native-nfc-manager
- **实时通信**: Socket.IO

## ⚠️ 遇到的问题

### 问题 1: 本地内存不足
**现象**: Gradle 守护进程反复崩溃  
**原因**: 服务器 2GB 内存，Android 编译需 4GB+  
**解决**: 暂停本地编译，等待公司环境

### 问题 2: Expo EAS 配置问题
**现象**: EAS Build 依赖安装失败  
**原因**: 项目是纯 React Native（非 Expo 托管）  
**解决**: 需要完整迁移到 Expo 或使用本地编译

### 问题 3: Gradle 版本兼容
**现象**: 插件 `com.facebook.react.settings` 无法解析  
**原因**: RN 0.73.6 与自动生成的 android 目录版本不匹配  
**解决**: 重新生成 RN 0.73.6 兼容的 android 目录

### 问题 4: 网络下载超时
**现象**: Gradle 依赖下载超时  
**原因**: 服务器网络不稳定  
**解决**: 配置国内镜像（未完成）
